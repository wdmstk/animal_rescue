## 1. サービス概要

### 1.1 「動物の救急手帳」とは

「動物の救急手帳」は、ペットの緊急時に必要な情報をスマートフォンで即座に確認できるサービスです。

**主な機能:**
- 緊急情報（持病・アレルギー・投薬）の登録
- QRコードによる緊急情報の共有
- 健康記録（ワクチン・投薬・検査）の管理
- 家族での情報共有

**料金:**
- 月額680円
- 30日間の無料トライアル

### 1.2 対応動物

- 犬
- 猫
- その他のペット（うさぎ、鳥、爬虫類など）

## 2. アカウント作成とログイン

### 2.1 アカウント作成

**手順:**

1. アプリケーションにアクセス
2. 「新規登録」をクリック
3. メールアドレスとパスワードを入力
4. 「登録」をクリック
5. 確認メールを開いてリンクをクリック

**注意:**
- パスワードは8文字以上
- メールアドレスは有効なものを使用

### 2.2 ログイン

**手順:**

1. アプリケーションにアクセス
2. 「ログイン」をクリック
3. メールアドレスとパスワードを入力
4. 「ログイン」をクリック

### 2.3 パスワードリセット

**手順:**

1. ログイン画面で「パスワードを忘れた」をクリック
2. メールアドレスを入力
3. 「リセットメールを送信」をクリック
4. メールに記載されたリンクからパスワードを再設定

## 3. ペットの登録

### 3.1 新しいペットの登録

**手順:**

1. ログイン後、ホーム画面で「ペットを追加」をクリック
2. 以下の情報を入力:
   - 名前
   - 種類（犬・猫・その他）
   - 品種
   - 性別
   - 誕生日（任意）
   - 体重（任意）
3. 「保存」をクリック

### 3.2 ペット情報の編集

**手順:**

1. ペット一覧から編集したいペットを選択
2. 「編集」をクリック
3. 情報を更新
4. 「保存」をクリック

### 3.3 ペットの写真の追加

**手順:**

1. ペット詳細画面で「写真を追加」をクリック
2. 写真を選択
3. 「アップロード」をクリック

**注意:**
- 対応形式: JPG, PNG
- 最大サイズ: 5MB

## 4. 緊急情報の設定

### 4.1 緊急情報の登録

**登録項目:**
- 持病・既往歴
- アレルギー
- 現在の投薬
- 血液型
- かかりつけ病院
- 緊急連絡先
- 夜間救急病院

**手順:**

1. ペット詳細画面で「緊急情報」をクリック
2. 各項目を入力
3. 「保存」をクリック

**重要性:**
- 緊急時に獣医師が参照する重要な情報です
- 正確な情報を登録してください

### 4.2 緊急連絡先の設定

**設定項目:**
- 緊急連絡先1（必須）
- 緊急連絡先2（任意）

**手順:**

1. 緊急情報画面で「緊急連絡先」を入力
2. 名前と電話番号を入力
3. 「保存」をクリック

**注意:**
- 緊急時に確実に連絡が取れる番号を登録
- 携帯電話の番号を推奨

## 5. 緊急QRコードの使い方

### 5.1 QRコードの表示

**手順:**

1. ペット詳細画面で「緊急QR」をクリック
2. QRコードが表示される

### 5.2 QRコードの使い方

**日常的な準備:**
- QRコードをスクリーンショットで保存
- スマートフォンの待ち受け画面に設定
- 首輪やタグにQRコードを印刷して取り付ける

**緊急時の使い方:**
1. 獣医師や救急隊員にQRコードを見せる
2. QRコードをスキャン
3. 緊急情報が表示される

### 5.3 緊急公開画面

**表示される情報:**
- ペットの名前
- 血液型
- 持病・アレルギー
- 現在の投薬
- 緊急連絡先（ワンタップで発信）
- かかりつけ病院（ワンタップで発信・地図）
- 夜間救急病院（ワンタップで発信・地図）

**特徴:**
- 認証不要で誰でも閲覧可能
- スマートフォンで最適化された表示
- 緊急時に重要な情報が優先的に表示

## 6. 健康記録の管理

### 6.1 ワクチン接種履歴

**手順:**

1. ペット詳細画面で「ワクチン」をクリック
2. 「追加」をクリック
3. 以下の情報を入力:
   - 種類（狂犬病・混合・その他）
   - 接種日
   - 次回接種日（任意）
4. 「保存」をクリック

### 6.2 投薬記録

**手順:**

1. ペット詳細画面で「投薬」をクリック
2. 「追加」をクリック
3. 以下の情報を入力:
   - 薬剤名
   - 用量
   - 頻度
   - 開始日
   - 終了日（任意）
4. 「保存」をクリック

### 6.3 医療記録

**手順:**

1. ペット詳細画面で「医療記録」をクリック
2. 「追加」をクリック
3. 以下の情報を入力:
   - 日付
   - タイトル
   - 詳細
   - 写真（任意）
4. 「保存」をクリック

## 7. 家族の招待

### 7.1 家族招待コードの発行

**手順:**

1. 設定画面で「家族」をクリック
2. 「招待コードを発行」をクリック
3. 有効期限を設定
4. 「発行」をクリック
5. 招待コードを家族に共有

### 7.2 家族として参加

**手順:**

1. 招待コードを受け取った家族がアプリにアクセス
2. ログイン後、設定画面で「家族招待コードを入力」をクリック
3. 招待コードを入力
4. 「参加」をクリック

### 7.3 権限管理

**権限の種類:**
- **OWNER**: 全ての機能を使用可能、招待コード発行、メンバー管理
- **FAMILY**: 閲覧・編集のみ、招待コード発行不可

**注意:**
- 最低1人のOWNERが必要
- 最後のOWNERは降格できない

## 8. 課金について

### 8.1 料金プラン

**無料トライアル:**
- 期間: 30日間
- 機能: 全ての機能を使用可能

**有料プラン:**
- 料金: 月額680円
- 支払い方法: クレジットカード
- 解約: いつでも可能

### 8.2 課金の開始

**手順:**

1. ログイン後、設定画面で「課金」をクリック
2. 「有料プランを開始」をクリック
3. 支払い情報を入力
4. 「登録」をクリック

### 8.3 解約方法

**手順:**

1. 設定画面で「課金」をクリック
2. 「解約」をクリック
3. 確認ダイアログで「解約」をクリック

**注意:**
- 解約後も支払い期間終了まで使用可能
- 解約してもデータは保持される

### 8.4 アカウント削除

**手順:**

1. 設定画面で「アカウント」をクリック
2. 「アカウントを削除」をクリック
3. 確認ダイアログで「削除」をクリック

**注意:**
- アカウント削除後、全てのデータが削除される
- 復元は不可能

## 9. よくある質問

### Q: QRコードはどこに保存すればいいですか？

A: 以下の場所に保存することをお勧めします:
- スマートフォンの待ち受け画面
- 首輪やタグに印刷して取り付ける
- お財布や鍵束に入れておく

### Q: 家族メンバーは何人まで招待できますか？

A: 現在、人数制限はありません。必要な人数を招待できます。

### Q: データは安全ですか？

A: はい、以下のセキュリティ対策を講じています:
- データは暗号化されて保存されます
- サーバーは安全的なデータセンターに配置されています
- 定期的なバックアップが実施されています

### Q: オフラインで使用できますか？

A: 現在、オフラインでの使用はサポートしていません。インターネット接続が必要です。

### Q: サポートに連絡するにはどうすればいいですか？

A: 設定画面の「サポート」から連絡フォームにアクセスしてください。
