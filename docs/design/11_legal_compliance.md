## 2. 個人情報保護法対応

### 個人情報の特定

```
個人情報（直接識別）:
  - メールアドレス
  - 表示名（氏名の場合）
  - 緊急連絡先名・電話番号
  - 飼い主住所（Pet.ownerAddress）

個人情報（間接識別・準個人情報）:
  - ユーザーID（auth.users.id）
  - デバイスID・IPアドレス（ログ）

センシティブ情報（要注意）:
  - ペットの医療情報（「個人の病歴」に準じた取扱いを推奨）
  ※ 法的には「ペットの情報」は個人情報でないが、
    飼い主との関連から個人情報に準じた管理が適切
```

### 取扱方針

```
利用目的の特定と明示:
  1. サービス提供（ペット管理・緊急情報共有）
  2. 本人確認・認証
  3. 課金・決済処理
  4. サービス改善・統計分析
  5. 障害調査・セキュリティ対応
  6. 問い合わせ対応

第三者提供:
  - 原則として本人同意なく第三者提供しない
  - 例外: 法令に基づく開示要請
  - 委託: Supabase（DB・Auth・Storage）, Stripe（決済）
    → 委託先との契約で個人情報の取扱いを規定

越境移転（Supabase/Stripeが海外サーバーを使用）:
  - プライバシーポリシーに移転先国・保護措置を明記
  - 個人情報保護法の「外国にある第三者への提供」規定に対応
```

### 本人の権利への対応

```
開示請求: メールで受付 → 本人確認後 → 14日以内に回答
訂正・削除請求: 設定画面でのセルフサービス + メール受付
利用停止: アカウント削除機能（未実装・要対応）
提供停止: 第三者提供の停止請求
```

## 3. 特定商取引法対応

### 記載必須事項（特商法11条）

```
必須記載項目:
  1. 販売事業者名: ○○株式会社（または個人事業主名）
  2. 運営統括責任者: 代表者名
  3. 所在地: 法人登記住所
  4. 連絡先: メールアドレス（電話番号は省略可・請求時に開示）
  5. 販売価格: 月額680円（税込）
  6. 商品代金以外の費用: なし（または通信費）
  7. 支払方法: クレジットカード
  8. 支払時期: 月次（トライアル終了後、毎月の契約更新日）
  9. 役務提供時期: 即時（課金成功後）
  10. 解約条件: Stripe Customer Portal または設定画面から解約可能
      解約後は期間終了まで利用可能
  11. 返金条件: 原則返金不可（例外あり）
  12. 動作環境: 最新版のChrome/Safari/Firefox、iOS 16+/Android 12+
```

### 特商法違反リスクと対策

```
リスク1: 解約手順が不明瞭
  対策: 設定画面に「解約」を明示的に配置（Stripe Customer Portal導線）

リスク2: 無料トライアルの条件が不明
  対策: 登録画面に「30日後から月額680円が課金されます」を明示

リスク3: 価格改定時の通知不足
  対策: 価格改定の30日前にメール通知 + アプリ内通知
```

## 4. 消費者契約法対応

### 問題になりやすい条項

```
【危険な免責条項の例（NGパターン）】
  × 「いかなる場合も責任を負いません」
  → 消費者契約法10条で無効になる可能性

【適切な免責条項の例（OKパターン）】
  ○ 「当社の故意・重過失による損害は賠償します。
     それ以外の損害は、1ヶ月分の利用料金を上限に賠償します」

【医療免責条項】
  ○ 「本サービスは医療情報の記録・管理ツールであり、
     獣医師による診断・治療の代替にはなりません。
     緊急時は必ず獣医師の判断に従ってください」
```

### 不当条項の排除チェックリスト

- [ ] 一方的な解約・変更条項はないか
- [ ] 消費者に不当に不利な免責はないか
- [ ] 準拠法・管轄裁判所は明示されているか（東京地方裁判所が一般的）
- [ ] 強行法規を排除していないか

## 5. 利用規約設計

### 必須条項

```
1. サービスの目的と提供条件
   「ペットの医療情報管理・緊急時情報共有のためのツール」

2. アカウント管理
   - ユーザーの責任でパスワードを管理
   - 第三者への貸与・譲渡禁止

3. 禁止事項
   - 他者の情報の不正アクセス
   - サービスの逆コンパイル・解析
   - 医療サービスとしての宣伝・使用
   - 商業目的での転売

4. 課金条項（重要）
   - 月額680円（税込）
   - 30日の無料トライアル
   - 自動更新（毎月）
   - 解約: 次の更新日の24時間前まで
   - 返金: 原則不可（未使用期間の比例返金は検討）

5. 免責事項
   - 医療助言・診断でないことの明示
   - 情報の正確性は飼い主の責任
   - 緊急時の情報提供の保証はしない（通信障害等）
   - 故意・重過失以外の損害の免責（限定賠償）

6. 知的財産
   - サービスのUI・コード・ロゴは事業者の知財
   - ユーザーが登録した情報はユーザーの所有
   - ユーザーはサービス内での表示に同意

7. サービスの変更・停止
   - 事前通知（30日前）での機能変更・価格改定
   - 重大な変更時はメール通知
   - サービス終了時のデータエクスポート機能の提供

8. 準拠法・管轄
   - 準拠法: 日本法
   - 管轄: 東京地方裁判所（専属合意管轄）
```

## 6. プライバシーポリシー設計

### 構造

```
1. 事業者情報
2. 収集する個人情報の種類
3. 利用目的
4. 第三者提供・委託
5. 越境移転
6. 保存期間
7. 本人の権利
8. Cookie・トラッキング
9. 未成年者への対応
10. ポリシーの変更
11. 問い合わせ先
```

### 緊急公開機能の特則（重要）

```
プライバシーポリシーに必ず記載:

「緊急公開機能について」
本サービスの緊急公開機能では、ユーザーが設定したQRコードを
利用することで、第三者がインターネット上でペットの緊急情報を
閲覧することができます。

公開される情報の範囲:
  - ペット名・種類・品種・年齢
  - 持病・アレルギー（ユーザーが入力した情報）
  - 現在の投薬情報
  - かかりつけ病院名・電話番号
  - 緊急連絡先（名前・電話番号）

公開の制御:
  - ユーザーがトークンを無効化することで即時非公開
  - QRコードを再生成することで旧URLを無効化

注意事項:
  - 公開URLは不特定多数がアクセス可能です
  - 共有には十分注意してください
  - 共有の責任はユーザーにあります
```

## 7. 医療行為・免責設計

### 獣医師法との関係

```
獣医師法との関係:
  第17条: 獣医師でない者は、獣医業を行ってはならない
  「獣医業」: 飼育動物の診療・治療を業として行うこと

本サービスの位置づけ:
  本サービスは「記録・管理・情報共有ツール」であり、
  診断・治療を提供するものではない

注意が必要な機能:
  - AI症状チェッカー（将来機能）
    → 「医療判断ではない」ことを明示する設計が必須
    → 「気になる場合は必ず獣医師に相談を」の導線を設ける
  
  - 薬の登録支援
    → 薬の用量・副作用に関する情報提供は獣医師法に抵触する可能性
    → ユーザーが入力した情報を表示するのみ（情報提供ではない）
```

### 誤認防止設計

```
絶対に避けるべき表現:
  × 「この薬が効きます」
  × 「○○の症状はこの病気です」
  × 「獣医師に代わって情報提供」
  × 「医療相談」

推奨する表現:
  ○ 「記録した情報を表示しています」
  ○ 「記録・管理ツール」
  ○ 「緊急時に情報をスムーズに共有するためのツール」
  ○ 「必ず獣医師の診断を受けてください」

UI上の免責表示（緊急公開画面）:
  ※このページは飼い主が登録した情報を表示しています。
  医療上の判断は必ず獣医師が行ってください。
  最終更新: YYYY-MM-DD
```

### AI機能の設計制約

```
許可される AI 活用（補助 AI）:
  ○ OCR（医療書類のテキスト抽出）
  ○ 薬剤名の読み取り補助（OCR精度向上）
  ○ 領収書・カルテのテキスト要約（飼い主の記録補助）
  ○ ワクチン証明書からの日付読取
  ○ 写真からのペット種類推定（オプション・エンタメ）

禁止される AI 活用（医療判断）:
  × 症状からの疾患推定
  × 薬の用量アドバイス
  × 「これは緊急です」の判定
  × 治療方針の提案
```

## 8. 同意管理設計

### 同意が必要なタイミング

```
アカウント登録時（必須同意）:
  □ 利用規約に同意する
  □ プライバシーポリシーに同意する
  □ 30日後から月額680円が課金されることを確認した

任意Cookie使用時（分析等）:
  □ 分析目的でCookieを使用することに同意する
  （チェックなし=拒否。サービスは引き続き利用可能）

同意記録:
  - 同意日時、IPアドレス、同意した規約バージョンをDBに記録
  - 規約改定時は再同意を求める
```

### 同意UI設計

```
NG（ダークパターン）:
  × 「同意する」のみのボタン
  × プリチェック済みの同意チェックボックス
  × 同意しないと画面が進めない（必須ではない同意）

OK（適切な設計）:
  ○ 「利用規約」「プライバシーポリシー」にリンク付き
  ○ チェックボックスはデフォルトで未チェック
  ○ 任意同意は明確に「任意」と表示
```

## 9. データ削除ポリシー

### 退会時のデータ削除

```
即時削除（退会後即座）:
  - アクティブセッション
  - QRトークン（公開URL無効化）
  - 認証情報（Supabase Auth）

猶予期間削除（30日後）:
  - ペット医療記録
  - 写真・書類
  - 設定情報
  猶予期間中は「アカウント復旧」が可能

法定保管（削除不可）:
  - 課金・会計記録（7年）※匿名化して保管
  - インシデント関連の監査ログ（3年）

削除リクエストの処理:
  1. 本人確認（メールでの確認リンク）
  2. 削除スケジュールの確認
  3. 14日以内に処理
  4. 完了通知メールを送信
```

## 10. Cookie・トラッキングポリシー

### Cookie分類

```
必須Cookie（同意不要）:
  - sb-*: Supabase 認証セッション
  - 役割: ログイン状態の維持

任意Cookie（同意必要）:
  - _ga, _gid: Google Analytics（将来導入の場合）
  - 役割: アクセス解析

ブロックされても機能するか:
  - 必須Cookie以外をブロック → サービスは正常に動作
```

## 11. その他法令対応

### 景品表示法

```
注意が必要な表現:
  × 「No.1の〇〇」（根拠なし）
  × 「完全無料」（トライアル後課金あり）
  × 「〇〇病院推薦」（事実でない場合）

適切な表現:
  ○ 「30日間の無料トライアル（その後月額680円）」
  ○ 「〇〇件のペットが登録されています」（事実の場合）
```

### 電気通信事業法

```
確認事項:
  ユーザー間のメッセージ機能がある場合 → 届出が必要
  本サービス: メッセージ機能なし → 現時点では対象外
  将来: 動物病院との連絡機能を追加する場合は要確認
```

## 12. リスクと代替案

### 主要法的リスク

| リスク | 深刻度 | 対応策 | 代替案 |
|---|---|---|---|
| 医療行為との誤認 | High | 免責表示の徹底 + 機能制限 | AI診断機能を完全排除 |
| 個人情報漏洩 | High | 暗号化・RLS・監査ログ | 情報最小化・仮名化 |
| 特商法表記不備 | High | 弁護士レビュー | 簡潔な表記で最低限対応 |
| 消費者トラブル（返金） | Medium | 返金ポリシーの明確化 | 一定条件で返金対応 |
| 未成年の利用 | Medium | 年齢確認・保護者同意 | 年齢制限（18歳以上）設定 |

### 推奨アクション（ローンチ前）

1. **弁護士レビュー依頼**: 利用規約・プライバシーポリシー・特商法表記
2. **医療免責の法的確認**: AI機能・症状記録機能の適法性確認
3. **個人情報保護委員会への相談**: 越境移転・センシティブデータの取扱い
4. **返金ポリシーの設計**: 消費者契約法リスクを考慮した設計
5. **Cookieポリシーの実装**: 同意バナーの実装
