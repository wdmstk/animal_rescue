## 現状把握サマリー（Phase 0）

### プロダクト概要
- **サービス名**: 動物の救急手帳（Emergency Pet Pass）
- **目的**: 緊急時に動物の命を守るための情報基盤
- **想定ユーザー**: 犬・猫・その他ペットの飼い主
- **提供形態**: SaaS（月額680円、30日無料トライアル）

### 技術スタック（現状）

| レイヤー | 技術 |
|---|---|
| Frontend | Next.js App Router + TypeScript |
| Backend | Next.js Route Handlers |
| Database | Supabase (PostgreSQL) |
| ORM | Prisma |
| Auth | Supabase Auth |
| Storage | Supabase Storage |
| Payment | Stripe |
| Styling | Tailwind CSS |
| Test | Vitest + Playwright |

- **TASK-101〜197**: 計97タスク中93タスクが done

- **主要機能実装済み**: ペット管理・緊急情報・QR公開・投薬管理・ワクチン・健康トラッキング・OCR・Stripe課金
